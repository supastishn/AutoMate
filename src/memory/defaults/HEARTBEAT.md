# HEARTBEAT.md - Proactive Tasks

_Things to check during periodic wakeups. Update this list as needed._

## On Each Heartbeat

- [ ] **Check goals** — Run `goals action=next`. If there's a pending goal, START IT with `goals action=start id="..."` then work on it
- [ ] Review today's daily log — anything worth promoting to MEMORY.md?
- [ ] Check if any cron jobs need attention
- [ ] Note anything the user might want to know about
- [ ] **Clean up** — Remove completed items from this list, mark done goals as complete

## Periodic (Weekly)

- [ ] Review MEMORY.md — remove outdated info, consolidate
- [ ] Check if USER.md needs updates based on recent interactions
- [ ] Review PERSONALITY.md — does it still feel right?
- [ ] Prune stale/obsolete goals with `goals action=delete`

## Autonomy Rules

1. If you see `[ ]` items above, DO THEM (or remove if obsolete)
2. If you complete a task, mark it `[x]` or remove it
3. If there are pending goals, pick one up and make progress
4. Don't just report — ACT
